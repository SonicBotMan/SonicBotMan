### 🔍 About Me

白天穿梭于需求评审和产品原型之间，深夜埋首于 TypeScript 和 Shell 脚本之中。

在北京互联网圈做产品经理的这几年，我逐渐意识到一个问题：**AI 工具越来越强，但把 AI 变成真正能用的产品，中间还隔着巨大的工程鸿沟。** 于是我没有等别人来解决——我自己动手，搭了一套以 OpenClaw 为核心的 AI 工作流基础设施。

我的项目不是玩具。从 35⭐ 的 [openclaw-portable](https://github.com/SonicBotMan/openclaw-portable) 到 28⭐ 的 [lobster-press](https://github.com/SonicBotMan/lobster-press)，再到最新的 [SoloFlow](https://github.com/SonicBotMan/SoloFlow) 工作流引擎——每一行代码都解决真实问题，每一行都有产品逻辑在背后支撑。

**这不是副业，这是我的第二产品线。**

## 📦 Core Projects

### [openclaw-portable](https://github.com/SonicBotMan/openclaw-portable) · 35 ⭐
> 跨平台 OpenClaw 即插即用解决方案

Shell 开发的便携式 AI Agent 运行环境，Windows/Linux/Mac 三端统一，开箱即用。

### [lobster-press](https://github.com/SonicBotMan/lobster-press) · 28 ⭐
> 🦞 龙虾饼 — 认知记忆系统

基于认知科学的 LLM 永久记忆引擎。语义 / 情景 / 工作三层记忆 + 遗忘曲线，让 AI Agent 真正"记住"。

### [SoloFlow](https://github.com/SonicBotMan/SoloFlow) · ⚡ NEW
> OpenClaw 工作流编排插件

将复杂多步骤 AI 任务转化为结构化、可观测、可重试的工作流。DAG+FSM 混合架构 + 认知记忆 + Skill 自动进化。
