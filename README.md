## 📦 Core Projects

### [openclaw-portable](https://github.com/SonicBotMan/openclaw-portable)
> 跨平台 OpenClaw 即插即用解决方案

Shell 驱动的便携式 AI Agent 运行环境，Windows / Linux / Mac 统一体验，开箱即用。

### [lobster-press](https://github.com/SonicBotMan/lobster-press)
> 🦞 龙虾饼 — 认知记忆系统

基于认知科学的 LLM 永久记忆引擎。语义 / 情景 / 工作三层记忆 + 遗忘曲线，让 Agent 真正「记住」。

### [ClawICU](https://github.com/SonicBotMan/clawicu) · 🏥 *OpenClaw ICU*
> 一条命令，跑完分诊 → 诊断 → 处置 → 复核

- **20** 个独立诊断模块（配置 JSON5、网关 `/healthz`、插件与 SDK、凭据、守护进程、版本不一致、`plugins.allow`、频道策略等）
- **6** 个阶段：Bootstrap → 带超时的 `openclaw doctor` → 检查合并分诊 → **可交互**处置菜单（支持 `curl | sh`）→ 修复与验证  
- **25** 篇线上故障指南 + Next.js 静态站、SEO、**SOS 分享页**（[xagent.icu/sos](https://xagent.icu/sos)）  

### [SoloFlow](https://github.com/SonicBotMan/SoloFlow) · ⚡
> OpenClaw 工作流编排插件

将复杂多步骤 AI 任务变成结构化、可观测、可重试的工作流。DAG + FSM 混合架构，面向生产。
